# Usage
<!-- usage -->
```sh-session
$ npm install -g expr-taskcli
$ etl COMMAND
running command...
$ etl (-v|--version|version)
expr-taskcli/0.6.8 darwin-x64 node-v14.15.0
$ etl --help [COMMAND]
USAGE
  $ etl COMMAND
...
```
